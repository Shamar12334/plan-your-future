# Plan-Your-Future

**Plan-Your-Future** is a web application that generates personalized career roadmaps for users based on the career they want to pursue. It uses Flask for the backend, React for the frontend, and JSON for data exchange.

---

## Features

- Generate detailed career roadmaps based on user input.
- User-friendly React interface.
- Flask backend API serving roadmap data in JSON format.
- Expandable roadmap templates for various careers.

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Flask (Python)
- **Data Exchange:** JSON
- **Optional:** Axios for API requests, Bootstrap/Material-UI for styling

---

## Installation

### Backend (Flask)

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/plan-your-future.git
    cd plan-your-future/backend
    ```

2. Create a virtual environment and install dependencies:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Linux/Mac
    venv\Scripts\activate     # Windows
    pip install -r requirements.txt
    ```

3. Run the Flask server:
    ```bash
    flask run
    ```

### Frontend (React)

1. Navigate to the frontend directory:
    ```bash
    cd ../frontend
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the React app:
    ```bash
    npm start
    ```

---

## Usage

1. Open the React app in your browser.
2. Enter the career you want to pursue.
3. The app will generate a detailed roadmap and display it on the page.
4. Save or export your roadmap (if feature is added).
